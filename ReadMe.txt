This is a C++ written package. 

It read raw data from json format txt file from folder \Data. The Data is from social network API.

Extract various relational inforamtion and store them in SQL data base. 

This package uses C++ ODBC and SQL server 2014. 

Various linked tables are used to save the relational inforamtion with minimum effort. 

Time efficiency of this package is O(N), with N as number of inputs.
